🔥 Belajar HTML Dasar untuk Pemula


✍️ Author: Anny Klaudya | 📅 4 September 2025

🏷️ Label:

  • 📘 HTML
  • 🌐 Web Development
  • 🛠️ Praktikum

📜 Deskripsi:

HTML (HyperText Markup Language) adalah fondasi utama dalam dunia web. Dengan HTML kita dapat membangun kerangka sebuah website.

Elemen dasar: <h1> untuk judul, <p> untuk paragraf, <img> untuk gambar, dan atribut seperti src serta href.

Setelah menguasai HTML, lanjutkan ke CSS untuk tampilan dan JavaScript untuk interaktivitas ✨.

⚡ Tips Produktif Belajar Pemrograman


✍️ Author: Anny Klaudya | 📅 1 September 2025

🏷️ Label:

  • 💡 Motivasi
  • 📖 Belajar
  • 💻 Teknologi

📜 Deskripsi:

Belajar pemrograman membutuhkan konsistensi dan strategi. Jangan hanya membaca teori, tapi juga sering praktik langsung.

Beberapa tips: buat jadwal belajar, mulai dari proyek kecil, gunakan dokumentasi resmi, dan jangan takut untuk gagal. Gagal adalah bagian dari proses.

Ingat, kunci produktif adalah fokus, latihan rutin, dan komunitas. Dengan itu, belajar jadi lebih menyenangkan 🚀.
